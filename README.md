# TRC: TeleportCompasses

A simple Paper plugin that adds a compass with the ability to teleport to a set location.

## Choose Version
Select the source code Version in the branch menu or download full build at:
- Modrinth (Paper): https://modrinth.com/plugin/trc-teleportcompasses
- Curseforge (Paper): https://www.curseforge.com/minecraft/bukkit-plugins/trc-teleportcompasses

## Available Versions
- 1.21.11
